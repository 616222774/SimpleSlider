SimpleSlider

A simple javascript slider without any dependencies on third-party libraries.



Example:


Usage

Just import the script on html and make sure it's available before starting it.

<div id="myslider" style="width:612px; height:612px">
    <img src="http://placekitten.com/g/612/612"/>
    <img src="http://placekitten.com/g/612/613"/>
    <img src="http://placekitten.com/g/612/614"/>
</div>
<script src="simpleslider.min.js"></script>
<script>
    var imgSlider = new SimpleSlider( document.getElementById('myslider') );
</script>

It only requires a group of elements inside of a main div tag (or similar). The slider element will always use overflow:hidden and its childs will have position:absolute.


More info

The goal of this small script is to enable an html element swapping adding the smallest piece of code possible.

The motivation to put this together is a frustration with such a simple functionality usually requiring plugins and frameworks.
