SimpleSlider

A simple javascript slider without any dependencies on third-party libraries.



Example:



Usage

Just import the script on html and make sure it's available before starting it.

<div id="myslider" style="width:612px; height:612px">
    <img src="http://placekitten.com/g/612/612"/>
    <img src="http://placekitten.com/g/612/613"/>
    <img src="http://placekitten.com/g/612/614"/>
</div>
<script src="simpleslider.min.js"></script>
<script>
    var imgSlider = new SimpleSlider( document.getElementById('myslider') );
</script>

It only requires a group of elements inside of a main div tag (or similar). The slider element will always use overflow:hidden and its childs will have position:absolute.



More info

The goal of this small script is to enable an carousel-like html element adding the smallest piece of code possible.

All the code is available on GitHub containing decent documentation, examples and unit tests. Please visit the project repository if you want to contribute or find more information about it.



This is so basic that it might just be an element

Precisely! The goal of the project is to be simple after all. To help with that, SimpleSlider also have an AngularJS directive (angular-simple-slider) and a Polymer element (polymer-simple-slider) implementations that just wrap the javascript into simpler html elements. Please visit these repositories for more information if you're interested in one of those.
